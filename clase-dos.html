Segunda Clase

Que es un Sistemas de Control de Versiones (VCS - Version Control System) de Código Fuente ?

El control de versiones se realiza durante el ciclo de desarrollo y vida del software, para controlar las distintas versiones del código fuente.Una versión, revisión o edición de un producto de software, es el estado en el que se encuentra el mismo en un momento dado de su desarrollo.

Características

  • Mecanismo de almacenamiento de los elementos que deba gestionar (ej. archivos de texto, imágenes, documentación...).
  • Posibilidad de realizar cambios sobre los elementos almacenados (ej. modificaciones parciales, añadir, borrar, renombrar o mover elementos).
  • Registro histórico de los cambios realizadas con cada elemento o conjunto de elementos (normalmente pudiendo volver o extraer un estado anterior del producto).

Terminología

La terminología empleada puede variar de sistema a sistema, pero a continuación se describen las mas comunes

  • Repositorio: El repositorio es el lugar en el que se almacenan los datos actualizados e históricos de cambios, a menudo en un servidor.
  • Revisión ("version"): Una revisión es una versión determinada de la información que se gestiona.
  • Rama ("branch") o fork : Un módulo puede ser branched o bifurcado en un instante de tiempo de forma que, desde ese momento en adelante se tienen dos copias (ramas) que evolucionan de forma independiente siguiendo su propia línea de desarrollo.
  • Desplegar ("Check-out", "checkout", "co") Un despliegue crea una copia de trabajo local desde el repositorio. Se puede especificar una revisión concreta, y predeterminadamente se suele obtener la última.

Las herramientas mas utilizadas en la actualidad son:

  • SVN - Apache Subversion (abreviado como SVN, por el comando svn) es una herramienta de control de versiones de Software Libre basada en un repositorio cuyo funcionamiento se asemeja enormemente al de un sistema de ficheros. - https://subversion.apache.org/
  • Git - Software de control de versiones diseñado por Linus Torvalds , - https://git-scm.com/
  • Otros Concurrent Versions System (CVS), Mercurial.
